# Day 07 – Pivot Tables in Excel

## What is a Pivot Table?

A Pivot Table is a powerful Excel feature used to summarize and analyze large datasets quickly.

It allows analysts to group data and calculate totals, averages, and counts.

---

## Example Dataset

| Order_ID | Product | Category | Sales |
|------|------|------|------|
| 101 | Laptop | Electronics | 55000 |
| 102 | Phone | Electronics | 18000 |
| 103 | Shoes | Fashion | 3000 |

---

## Example Analysis

Total Sales by Category:

| Category | Total Sales |
|------|------|
| Electronics | 75500 |
| Fashion | 7000 |

Pivot tables help analysts quickly summarize business data and generate insights.

---

## Key Steps

1. Select dataset
2. Go to Insert → Pivot Table
3. Drag fields into Rows and Values sections
