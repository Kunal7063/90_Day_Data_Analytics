# Data Analytics Learning Journey (90 Days)

This repository documents my 90-day journey of learning Data Analytics.

The goal of this project is to build strong analytical skills and develop a portfolio demonstrating practical knowledge of:

- Excel
- SQL
- Python
- Data Visualisation
- Business Case Studies

## Learning Structure

| Phase | Topics |
|------|------|
| Phase 1 | Data Analytics Fundamentals |
| Phase 2 | Excel for Data Analysis |
| Phase 3 | SQL for Data Analytics |
| Phase 4 | Python (Pandas, NumPy, EDA) |
| Phase 5 | Data Visualization |
| Phase 6 | Real-world Data Projects |

## Daily Progress

- Day 01 – Introduction to Data Analytics
- Day 02 – Types of Data

## Goal

Become job-ready for a Data Analyst role within 6 months by building strong practical skills and real-world projects.
