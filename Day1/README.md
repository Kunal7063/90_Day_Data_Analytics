Day 01 – Introduction to Data Analytics
📌 What is Data?

Data is raw facts or information collected for analysis.
Examples include customer age, product price, sales numbers, website visits, etc.

Raw data alone does not provide value until it is analyzed.

📌 What is Data Analytics?

Data Analytics is the process of:

Collecting data

Cleaning data

Analyzing data

Identifying patterns

Making data-driven decisions

In simple terms, it means converting raw data into meaningful insights.

📌 Types of Data Analytics
1. Descriptive Analytics

Answers: What happened?
Example: Total sales last month were ₹5,00,000.

2. Diagnostic Analytics

Answers: Why did it happen?
Example: Sales dropped due to reduced website traffic.

3. Predictive Analytics

Answers: What is likely to happen?
Example: Sales may increase during the festive season.

4. Prescriptive Analytics

Answers: What should be done?
Example: Offer weekend discounts to increase sales.

📌 Role of a Data Analyst

A Data Analyst:

Collects and cleans data

Uses tools like Excel, SQL, and Python

Creates dashboards and reports

Provides insights to support business decisions

A Data Analyst helps organizations make informed decisions using data.
