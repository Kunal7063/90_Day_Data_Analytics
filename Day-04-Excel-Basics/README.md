# Day 04 – Excel Basics for Data Analytics

## Role of Excel in Data Analytics

Excel is widely used by data analysts for:
- Data cleaning
- Sorting and filtering
- Quick calculations
- Data summarization
- Creating reports

---

## Dataset Structure

A good dataset follows a tabular structure.

Example:

| Order_ID | Customer | Product | Price | Quantity |
|---------|---------|--------|------|---------|
| 101 | Rahul | Laptop | 55000 | 1 |
| 102 | Sneha | Shoes | 3000 | 2 |

Rules:
- First row contains column headers
- Each row represents one record
- Each column represents one variable

---

## Sorting Data

Sorting organizes data in ascending or descending order.

Example: Sorting products by price helps identify the most expensive or cheapest items.

---

## Filtering Data

Filtering allows analysts to display only specific rows based on conditions.

Example: Showing only Electronics category products.

---

## Basic Calculation

Example formula:

Total Sales = Price × Quantity

Excel formula:

=B2*C2
