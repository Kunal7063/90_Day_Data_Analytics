# Day 12 – VLOOKUP in Excel

## What is VLOOKUP?

VLOOKUP is used to search for a value in a table and return corresponding data.

## Syntax

=VLOOKUP(lookup_value, table_array, col_index, FALSE)

## Example

=VLOOKUP(A2, D2:E5, 2, FALSE)

## Use Cases

- Fetch product prices
- Merge data from multiple tables
- Data mapping

VLOOKUP is widely used in real-world data analysis.
