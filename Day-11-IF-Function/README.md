# Day 11 – IF Function in Excel

## What is IF Function?

The IF function is used to make decisions based on conditions.

## Syntax

=IF(condition, value_if_true, value_if_false)

## Example

=IF(A1>10000,"High","Low")

## Nested IF

Used for multiple conditions.

Example:

=IF(A1>20000,"Top",IF(A1>=5000,"Medium","Low"))

## Use Cases

- Categorizing sales performance
- Decision making
- Data classification

The IF function is widely used in data analysis.
