# Day 06 – Data Cleaning in Excel

## Removing Duplicates

Duplicate records can affect analysis accuracy.

Excel provides a Remove Duplicates feature in the Data tab to eliminate repeated rows.

---

## Handling Missing Values

Datasets sometimes contain blank or missing values.

Common solutions:
- Fill with average values
- Replace with a suitable value
- Remove the record

---

## TRIM Function

The TRIM function removes extra spaces from text.

Example:

=TRIM(A2)

This ensures consistent text formatting.

---

## Text to Columns

Text to Columns allows splitting data into multiple columns.

Example:

Full Name → First Name + Last Name

This helps structure text data properly.
