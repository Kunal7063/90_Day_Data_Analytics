# Day 09 – Filtering and Sorting in Excel

## What is Filtering?

Filtering allows us to view only specific data based on conditions.

Example:
Filter only Electronics products from a dataset.

---

## What is Sorting?

Sorting arranges data in a specific order.

Types:
- Ascending (small to large)
- Descending (large to small)

---

## Use Cases

- Identify top selling products
- Find low performing categories
- Analyze specific data quickly

---

Filtering and sorting are essential skills for data analysis in Excel.
