# Day 14 – INDEX and MATCH in Excel

## What is INDEX?

Returns a value from a specific position in a range.

## What is MATCH?

Finds the position of a value in a range.

## Combined Use

=INDEX(B2:B5, MATCH(A2, A2:A5, 0))

## Use Cases

- Advanced lookup operations
- Flexible data searching
- Alternative to VLOOKUP

INDEX + MATCH is widely used in real-world data analysis.
