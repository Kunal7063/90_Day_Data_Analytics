# Day 05 – Important Excel Functions

## SUM Function

Used to calculate total values.

Example:

=SUM(B2:B6)

This adds all values in the selected range.

---

## AVERAGE Function

Used to calculate the mean value of numbers.

Example:

=AVERAGE(B2:B6)

This returns the average sales value.

---

## COUNT Function

Counts numeric values in a dataset.

Example:

=COUNT(B2:B6)

---

## IF Function

Used to perform logical tests.

Example:

=IF(B2>10000,"High","Low")

This categorizes sales values as High or Low.

---

## VLOOKUP

Used to retrieve information from another table.

Example:

=VLOOKUP(A2,E2:F6,2,FALSE)

This finds a value in another dataset and returns related information.
