# Day 10 – Excel Formulas for Data Analysis

## Important Formulas

### COUNT()
Counts the number of numeric values in a dataset.

### AVERAGE()
Calculates the average of values.

### MAX()
Finds the highest value.

### MIN()
Finds the lowest value.

---

## Use Cases

- COUNT → total records
- AVERAGE → average sales
- MAX → highest selling product
- MIN → lowest selling product

---

These formulas are essential for data analysis in Excel.
